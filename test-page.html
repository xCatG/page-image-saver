Instructions

To test the extension:

  1. Click the extension icon in your browser toolbar
  2. The extension should display a sidebar with the images found on this page
  3. You should see the large images and background images, but possibly not the small icons
  4. If configured correctly, you'll be able to select and save images to your storage
